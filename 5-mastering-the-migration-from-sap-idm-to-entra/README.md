# Mastering the Migration: A technical Deep Dive into Moving from SAP Identity Management (IDM) to Microsoft Entra
## 1. Introduction
Welcome to the DSAG TechXChange 2025 Hands-on lab. This document provides an overview of the Hands-on activities, challenges, and resources available to participants.

In this hands-on session you will implement a comprehensive solution for providing users with access to sign-in to SAP applications. You'll learn how to integrate Microsoft Entra with SAP Cloud Identity Services and leverage Microsoft Entra's advanced features to migrate and modernize existing SAP IDM approvals workflows, rules- and roles-based provisioning, and user self-services.
